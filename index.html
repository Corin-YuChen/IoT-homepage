課程大綱 (簡易版)

週次 主題 簡介
1 物聯網與雲端平台介紹 介紹物聯網、感測器網路及雲端數據分析平台的建構。
2 機率與統計基礎 複習數據分析所需的機率與統計基本概念。
3 Python 數據科學程式設計 學習使用 Python 進行數據科學程式設計的基礎。
4 資料預處理 (Pandas) 學習使用 Pandas 套件進行資料的清洗、轉換與整理。
5 監督式機器學習 介紹決策樹、迴歸、分類、集成模型等監督式學習演算法。
6 Python 機器學習模型建置 學習在 Python 中建立預測模型。
7 案例研究 透過實際案例，應用所學知識解決問題。
8 期末專題 分組進行物聯網與商業智慧相關的期末專題。
